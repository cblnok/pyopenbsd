The KQueue class represents a single kernel event queue.

Interface

KQueue()

The KQueue class requires no instantiation parameters.

kevent(changelist=None, nevents=0, timeout=None)

The kevent method is used to register or retrieve events. It accepts the following parameters:

  • changelist - a list of KEvents defining a set of changes to the queue.
  • nevents - the number of events to retrieve.
  • timout - maximum interval to wait for an event. If timeout is None, kevent waits indefinitely.
